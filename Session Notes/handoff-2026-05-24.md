## What shipped today

### Phase 1 — Foundation (continuation from 2026-05-23 scaffold)
- Repo + Vercel deploy + Neon Postgres + Resend wired through
- Multi-user signup gated by `SIGNUP_KEY` env var
- Email + password auth (bcrypt, HMAC-signed session cookies)
- Edge-safe middleware (Web Crypto for HMAC; client/server split for db modules)
- Custom domain `hub.farleycreative.com` live
- Light/dark theme + accent color
- Agent access (per-token Bearer auth, hashed at rest)
- MCP server (`/api/mcp`) with HTTP transport

### Brand
- Studio brand kit (pre-populated with Farley Girls Creative + her URLs)
- Client brand kits (multi-row, full CRUD UI, color swatches)
- Brand book PDF upload + auto-extraction to brand_book_notes
- Brand identity propagates: layout reads `--accent` CSS variable, `hub_label` drives titles

### Assets
- Vercel Blob storage integration (public access)
- Asset library at `/assets` — upload, list (grid with thumbnails), kind classification, link to brand kit, delete
- Image thumbnails inline; 📄 for PDFs; 📦 for other

### Drafts
- AI drafting via Claude Sonnet 4.5
- Brand-aware system prompt (cached for ~90% cost reduction on the brand block)
- Per-kind output guidance (listing has Title/Description/Tags; pin has Title/Description; email has Subject/Body; etc.)
- Voice picker — draft in any brand kit's voice (studio or client)
- Status workflow: draft → approved → published → archived
- Prospect-aware drafts: when `prospect_id` is set, Claude gets prospect context as a second (uncached) system block + activity auto-logged

### Pipeline (sales)
- **Active prospects** (`/pipeline`):
  - Full CRUD: business_name, industry, size, city/state (50 + DC), service_interest multi-select, status, next_action + date, source, notes
  - Contacts per prospect with primary-contact promotion
  - Activity timeline (email_sent/call/meeting/proposal_sent/note/status_change)
  - Drafts-for-this-prospect card (inline create with voice picker)
  - Promote-to-client-kit button (auto-creates brand_kits row, links back via `from_prospect_id`, flips status to signed, logs activity)
- **Leads** (`/pipeline/leads`):
  - Upstream queue — job postings, RFPs, articles, social mentions, referrals, cold-list entries
  - **AI parser**: paste posting/URL, Claude extracts business_name, location, industry, size, service_signal + summary; auto-populates the form
  - Convert-to-prospect with source attribution preserved in prospect notes + activity log
- Sub-nav: Active prospects | Leads

### Hub home
- Time-aware greeting (browser local time)
- **Awaiting You** — overdue prospect actions (red dot) + open drafts, combined count
- **Today** — 24h activity summary
- **Quick actions** — links with live counts
- **Recent activity feed** — cross-surface chronological list (15 events from drafts/prospects/activity/leads/assets, merged TS-side)
- **Recent assets** strip

### Etsy OAuth scaffold (ready to activate)
- Full v3 OAuth with PKCE — connect/callback/disconnect/test routes
- Auto-refresh client (`etsyFetch`) with 5-min refresh window
- `/settings/etsy` page with three states (not configured / not connected / connected)
- Will activate the moment `ETSY_CLIENT_ID` + `ETSY_CLIENT_SECRET` are set in Vercel + Etsy approves the app

### Cadence schema (stub for next session)
